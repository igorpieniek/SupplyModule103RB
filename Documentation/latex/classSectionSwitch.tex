\doxysubsection{Detailed Description}
Tools for switching ON / O\+FF specific supply branch. It help in easy way to set branch in right state. It solve the problem of branch state -\/ when pin is set to O\+FF but branch is still ON because of mechanical slider switch -\/\mbox{\hyperlink{classSectionSwitch_a5eb1dfe3252aa7e6db55989f38b4ce89}{update\+Real\+State\+O\+N()}}

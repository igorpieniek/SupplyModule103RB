Detailed Description

Tool to help with temperature sensor calculations. Use datasheet formula :
T = ( [Vsense - V25] / Avg_slope] + 25
where:
Vsense = current sensor voltage
V25 = TEMPERATURE_V25
Avg_slope = TEMPETATURE_AVG_SLOPE

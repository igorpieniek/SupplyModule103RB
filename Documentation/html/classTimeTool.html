Detailed Description

Tool to measure time in miliseconds. It provide function to get current CNT register state and to compare between many measurments - compareMicros(). It use timer TIMETOOL_TIMER

Warning
#1 Only one global object could be created, but no more!
#2 MX_TIMx_Init() have to be called before init() method
